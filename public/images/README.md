# Project Screenshots

Place your project screenshots in this directory with the following names:

## Required Images:

1. **krishilink-screenshot.png** - Screenshot of KrishiLink homepage
2. **scholarstream-screenshot.png** - Screenshot of ScholarStream homepage  
3. **toytopia-screenshot.png** - Screenshot of ToyTopia homepage
4. **heroapps-screenshot.png** - Screenshot of Hero Apps homepage

## Image Guidelines:

- **Format:** PNG or JPG
- **Dimensions:** 1200x750px (16:10 aspect ratio) recommended
- **Quality:** High resolution for crisp display
- **Content:** Show the main homepage/dashboard of each project
- **File Size:** Keep under 500KB for optimal loading

## How to Add Images:

1. Take screenshots of your deployed projects
2. Resize them to 1200x750px (or maintain 16:10 ratio)
3. Save them with the exact filenames listed above
4. Place them in this `/public/images/` directory

The portfolio will automatically display these images with a fallback to gradient backgrounds if images are not found.
